To run the algorith, type 'run_ga' into the console while this folder is set as
the root folder in matlab's current folder. run_ga.m is a script to call the
solution function that does the algorithm given certain parameters.

b) Fitness function is equal to the negative of ISE. ISE is to be minimized, and
ISE is always positive. Fitness function is made so that the higher the fitness,
the better the odds of survival, so the negative of ISE represents the fitness
(0 is the best and ISE of infinity (meaning fitness of -infinity) is the worst).
